A# Giraffe Guardians - IRGCP

This is a NextJS application for the Isiolo Reticulated Giraffe Conservation Program (IRGCP).

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.

## Tech Stack

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **UI Components**: ShadCN UI
- **Icons**: Lucide React
- **AI Integration**: Genkit with Google AI plugin

## Moving to GitHub

To move this project to your GitHub repository, follow these steps:

1. Create a new repository on GitHub.
2. Open your terminal in the project root.
3.<> Initialize git:
   ```bash
   git init
   ```
4. Add all files:
   ```bash
   git add .
   ```
5. Commit your changes:
   ```bash
   git commit -m "Initial commit from Firebase Studio"
   ```
6. Link to your GitHub repository:
   ```bash
   git remote add origin <your-repository-url>
   ```
7. Push to GitHub:
   ```bash
   git push -u origin main
   ```

This project is configured for Firebase App Hosting. Check `apphosting.yaml` for configuration details.
