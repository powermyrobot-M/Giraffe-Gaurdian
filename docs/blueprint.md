# **App Name**: Giraffe Guardians

## Core Features:

- Homepage Hero Section: Display a compelling hero section with a large heading, subtitle, and introductory paragraph about the IRGCP.
- Informational Sections: Present 'Our Mission', 'Protection Efforts', 'Community Engagement', and 'Education Programs' sections to inform visitors about the organization's work.
- Giraffe Gallery: Showcase a responsive grid of reticulated giraffe images with captions.
- Donation Call-to-Action: Include a prominent 'Your Donation Makes a Difference' section with a clear 'Donate Now' button.
- M-Changa Integration: Embed or link to an M-Changa fundraiser page with instructions for donation.
- M-Pesa PayBill Integration: Display PayBill number, account number, and instructions for donating via M-Pesa.
- Donation Amount Suggestion: Generative AI will suggest a donation amount based on what portion of the funding is still needed for the month. The LLM acts as a tool deciding what, if anything, it should display in this space.

## Style Guidelines:

- Primary color: Earthy green (#8FBC8F), evoking nature and conservation.
- Background color: Light desaturated beige (#F5F5DC) creating a clean, neutral backdrop.
- Accent color: Muted olive (#6B8E23), drawing visual interest without being overwhelming.
- Body font: 'PT Sans', a modern humanist sans-serif font that is suitable for body text.
- Headline font: 'Space Grotesk', a proportional sans-serif font with a computerized, techy, scientific feel. Suitable for headlines.
- Use simple, outlined icons to represent key actions and categories.
- A clean, single-page layout with clear sections and a prominent donation CTA.
- Subtle transitions and hover effects for a smooth user experience.
